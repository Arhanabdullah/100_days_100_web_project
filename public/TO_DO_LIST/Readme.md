# 📋 JUST DO IT — To-Do List

A sleek, modern, fully client-side to-do list web application built with HTML, CSS, and JavaScript. Features a stunning glassmorphism UI with floating animated shapes, theme switching, category management, PDF export, and a real-time progress tracker.

---

## ✨ Features

| Feature | Description |
|---|---|
| **➕ Add Tasks** | Enter a task with an optional category and add it instantly |
| **📂 Categories** | Work 💼, Personal 🧑, Urgent ⚡, Fitness 💪, Miscellaneous 📌 |
| **🏷️ Category Badges** | Each task card shows a colour-coded emoji badge for its category |
| **✅ Complete / Delete** | Mark tasks as done (animated strikethrough) or remove them |
| **✏️ Editable Text** | Click any task text to edit it inline |
| **🎨 5 Themes** | Switch between Purple, Pastel, Green, Dark, and Red themes |
| **📊 Progress Bar** | Live "X / Y done" tracker with animated fill |
| **📥 PDF Export** | Save your task list as a styled PDF via jsPDF |
| **📁 Document History** | View, download, or delete saved PDF documents |
| **🔔 Toast Notifications** | Confirmation popup when PDF is created |
| **✨ Micro-animations** | Staggered fade-in, hover lifts, smooth transitions |
| **📱 Fully Responsive** | Works beautifully on mobile, tablet, and desktop |
| **🌀 Floating Shapes** | Animated gradient orbs in the background |
| **🔲 Glassmorphism** | Frosted-glass panels with backdrop-blur |
| **📄 Empty States** | Friendly messages when no tasks or documents exist |

---

## 🛠️ Technologies Used

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, glassmorphism, keyframe animations, responsive media queries
- **JavaScript (Vanilla JS)** — All application logic, DOM manipulation, event handling
- **jsPDF** — Client-side PDF generation
- **Google Fonts (Inter)** — Modern typeface

## ▶️ How to Run

1. **Clone the repository**  
   ```bash
   git clone https://github.com/Dipayan-max/100_days_100_web_project.git
   ```

2. **Navigate to the project**  
   ```bash
   cd 100_days_100_web_project/public/TO_DO_LIST
   ```

3. **Open in browser**  
   Simply open `todolist.html` in any modern browser (no build tools needed).

4. **Start managing your tasks!**
